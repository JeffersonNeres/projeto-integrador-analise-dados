# Projeto Integrador - Análise de Dados

## Tema do Projeto
Análise de vendas de uma empresa de varejo.

## Integrante
Jefferson Neres da Silva

## Objetivo da Análise
Analisar o desempenho de vendas por produto, cidade e vendedor para identificar padrões e oportunidades de crescimento.

## Planejamento das Tarefas
1. Escolha da base de dados
2. Limpeza e tratamento dos dados
3. Análise exploratória
4. Criação das métricas
5. Construção do dashboard
6. Documentação do projeto

## Ideia Inicial do Dashboard

O dashboard apresentará:

- faturamento total
- vendas por cidade
- vendas por produto
- vendas por vendedor
- evolução das vendas ao longo do tempo
